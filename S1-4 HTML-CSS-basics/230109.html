섹션3 : HTML,CSS 추가

Live server extention을 이용해서 저장할 때마다 새로고침하게 만들 수 있다.

  1. ol list는 순서가 붙어있다 = numbers
  2. ul list는 순서가 없다 = bullet point
  3. 새롭게 안 사실 : 같은 항목에 두가지 동일한 스타일을 지정했을 때는 마지막 CSS가 적용된다 (파이썬과 반대)
  4. display: inline을 통해 버튼처럼 글자를 만들 수 있다.
  5. display: inline-blcok를 이용하면 inline 요소에도 위아래 여백을 잘 끼워줄 수 있다.
  6. 상대경로 : html위치가 더 깊숙하면 보기가 힘듦
  7. 절대경로 : root 부터 다시 검색함, 상대경로에서 /을 맨앞에 붙여주면 됨
